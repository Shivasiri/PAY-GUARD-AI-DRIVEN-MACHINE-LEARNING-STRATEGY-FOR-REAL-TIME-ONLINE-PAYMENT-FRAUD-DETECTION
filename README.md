# Pay Guard – AI Driven Machine Learning Strategy for Real-Time Online Payment Fraud Detection

## 📌 Project Overview
Pay Guard is an AI-driven fraud detection system designed to identify fraudulent online payment transactions in real time. The system uses Machine Learning algorithms to analyze transaction patterns, detect anomalies, and improve payment security. The project focuses on increasing fraud detection accuracy while reducing false alerts and improving transaction safety.

---

## 🚀 Features
- Real-time fraud detection
- Machine Learning based prediction
- Data preprocessing and feature engineering
- Multiple ML models implementation
- Fraud alert and notification system
- User/Admin dashboard output
- Secure transaction monitoring
- Prediction result storage
